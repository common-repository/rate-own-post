Rate Own Post plugin helps to add ratings for you post on your own.

== Description ==

Rate Own Post is a plugin which adds new column ratings on your *post edit* page,
 where you can have your own ratings for you own post.

* Use shortcode tag `[nyrating]` to show 5 star rating box, which you can see on
screenshots tab.


== Installation ==

This plugin is installed just like any other WordPress plugin. 
More [detailed installation instructions](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins "Installing Plugins - WordPress Codex") are available on the WordPress Codex.

If you want manual installation:

1. Download Rate Own Post
1. Extract `rate-own-post` folder.
1. Upload this folder with files inside to `../wp-contents/plugins/`
1. Activate the plugin in *Dashboard > Plugins*
1. Use shortcodes for require features.
1. Enjoy the ratings. And then consider donating.
